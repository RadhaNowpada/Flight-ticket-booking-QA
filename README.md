# Flight-ticket-booking-QA
Manual QA project for a Flight Ticket Booking application including requirements, test scenarios, test cases, defect reports, and SQL queries.
Flight Ticket Booking - Manual QA Project
This repository contains a demo Manual QA project for a Flight Ticket Booking web application.
## Scope
- Requirement Analysis
- Test Scenario Creation
- Test Case Design & Execution
- Defect Reporting
- Backend Data Validation using SQL
- ## Modules Covered
- Search Flights
- Flight Selection
- Passenger Details
- Payment
- Booking Confirmation

## Tools Used
- Manual Testing
- SQL (MySQL)
- Spreadsheets (Excel / Google Sheets)
- GitHub

This project is created for learning and demonstration purposes.
