## 1. Search Flights
- User should be able to select source and destination cities
- User should select departure date
- System should display available flights based on search criteria
- System should show error message for invalid inputs

## 2. Flight Selection
- User should be able to select a flight from the available list
- Fare details should be displayed correctly
- Selected flight details should persist to next page

## 3. Passenger Details
- User should enter passenger information such as name, gender, and date of birth
- Mandatory fields should be validated
- System should allow multiple passengers as selected

## 4. Payment
- User should be able to choose payment method
- Booking should be successful only after successful payment
- Failed payment should not create booking

## 5. Booking Confirmation
- System should generate a booking ID (PNR)
- Booking details should be displayed on confirmation page
